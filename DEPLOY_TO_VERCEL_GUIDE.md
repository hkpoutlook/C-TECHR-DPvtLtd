## **STEP 1: Prepare Your Code (5 minutes)**

### Option A: If you have GitHub account
1. Go to https://github.com
2. Create new repository named `ctech-rd`
3. Clone it locally

### Option B: If you don't have GitHub
1. Create GitHub account at https://github.com
2. Follow Option A above

---

## **STEP 2: Deploy to Vercel (5 minutes)**

### Instructions:

1. **Go to Vercel**
   - Visit https://vercel.com
   - Click "Sign Up"
   - Choose "Continue with GitHub"

2. **Authorize Vercel**
   - Click "Authorize Vercel"
   - Vercel will connect to your GitHub

3. **Import Project**
   - Click "New Project"
   - Select your `ctech-rd` repository
   - Click "Import"

4. **Configure Project**
   - Framework: React
   - Root Directory: `website/frontend`
   - Build Command: `npm run build`
   - Output Directory: `build`
   - Click "Deploy"

5. **Wait for Deployment**
   - Vercel will build and deploy
   - Takes 2-5 minutes
   - You'll get a URL like: `https://ctech-rd.vercel.app`

## **STEP 3: Get Free Domain (5 minutes)**

### From Freenom:

1. **Go to Freenom**
   - Visit https://www.freenom.com
   - Click "Find a new domain"

2. **Search Domain**
   - Type: `ctech-rd`
   - Choose extension: `.tk` (or `.ml`, `.ga`, `.cf`)
   - Click "Check Availability"

3. **Register Domain**
   - Click "Get it now!"
   - Select "12 Months FREE"
   - Click "Continue"
   - Complete checkout (free)

4. **Verify Email**
   - Check your email
   - Click verification link
   - Domain is now yours!

---

## **STEP 4: Connect Domain to Vercel (10 minutes)**

### In Vercel Dashboard:

1. **Go to Project Settings**
   - Open your Vercel project
   - Click "Settings"
   - Click "Domains"

2. **Add Domain**
   - Click "Add Domain"
   - Enter your Freenom domain (e.g., `ctech-rd.tk`)
   - Click "Add"

3. **Configure DNS**
   - Vercel will show nameservers
   - Copy the nameservers

### In Freenom Dashboard:

1. **Go to My Domains**
   - Login to freenom.com
   - Click "My Domains"
   - Click your domain

2. **Manage Domain**
   - Click "Manage Domain"
   - Click "Management Tools"
   - Click "Nameservers"

3. **Update Nameservers**
   - Select "Use custom nameservers"
   - Paste Vercel's nameservers
   - Click "Change Nameservers"

4. **Wait for DNS**
   - DNS propagation takes 24-48 hours
   - Your domain will work after that

## **Troubleshooting**

### Domain not working after 24 hours?
1. Clear browser cache
2. Try incognito mode
3. Wait another 24 hours
4. Check DNS settings in Freenom

### Build failed on Vercel?
1. Check build logs in Vercel
2. Ensure `package.json` has correct scripts
3. Verify all dependencies are listed

### Website looks different?
1. Hard refresh (Ctrl+F5)
2. Clear browser cache
3. Try different browser
