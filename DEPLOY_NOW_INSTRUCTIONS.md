## **Troubleshooting**

### Build Failed?
- Check Vercel build logs
- Ensure `package.json` has correct scripts
- Verify all dependencies listed

### Website Looks Wrong?
- Hard refresh (Ctrl+F5)
- Clear browser cache
- Try incognito mode

### Domain Not Working?
- Wait 24-48 hours for DNS
- Check nameservers in Freenom
- Verify settings in Vercel
